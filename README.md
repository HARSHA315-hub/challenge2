Halleyx Dashboard Builder:

A full-stack Custom Dashboard Builder that allows users to create, manage, and visualize business data using dynamic widgets such as charts, tables, and KPI cards.
Built with React (Frontend) and Django REST Framework (Backend), this project demonstrates real-time data handling, modular UI design, and scalable architecture.

Features:

Dynamic Dashboard Builder
Drag-and-drop layout system
Responsive grid (12/8/4 columns)
Save & load custom dashboard layouts

Order Management System:

Create, update, delete customer orders
Real-time data integration with dashboard
Automatic revenue calculation

Widgets Supported:

KPI Cards (Revenue, Orders, Customers)
Bar Chart
Line Chart
Pie Chart
Area Chart
Scatter Plot
Data Tables

Modern UI/UX:

Glassmorphism design
Gradient-based themes
Fully responsive layout

Tech Stack:
Frontend:

React (Vite)
Tailwind CSS
Recharts (for charts)

Axios (API calls):
Backend:

Django
Django REST Framework
PostgreSQL
